# cmd_tools

Linux command line tools

## Setup

```yaml
dependencies:
  tekartik_cmd_tools:
    git:
      url: https://github.com/tekartikprj/io_tools.dart
      path: packages/cmd_tools
      ref: dart3a
    version: '>=0.2.0'
```
## kill_cmd

kill all chrome instance

    kill_cmd chrome
    
Kill qemu

    kill_cmd qemu
